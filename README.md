# Typing Practice App

**TypingPractice** is a minimal, sleek typing practice application built with **React**, **TypeScript**, and **Tailwind CSS**. It helps users improve typing speed, accuracy, and rhythm in a focused environment.

## Features

- Adjustable timer: 20s, 30s, 40s, 50s, 60s, 90s, 120s  
- Countdown before typing starts  
- Real-time character-level correctness highlighting  
- Session summary with:
  - WPM (Words per Minute)
  - Accuracy
  - Correct / Wrong characters
- Simple, professional dark-themed design  
- Desktop and laptops only  

## Live Demo

[View Live on Vercel](https://typingpractice-xi.vercel.app/)  

## Installation (for development)

```bash
# Clone the repository
git clone https://github.com/saadhumayun012/typingpractice.git

# Install dependencies
npm install

# Start development server
npm run start
````

## Usage

1. Select a timer duration
2. Click **Start Practice**
3. Type the text that appears, focusing on speed and accuracy
4. View results after session ends
